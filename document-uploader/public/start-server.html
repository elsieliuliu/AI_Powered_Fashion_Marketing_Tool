Manual Server Start Instructions

If the automatic server start doesn't work, please follow these steps:

  1. Open a terminal or command prompt
  2. Navigate to the server directory:
    cd document-uploader/server
  3. Start the server:
    node server.js
  4. You should see a message showing which port the server is running on
  5. Return to the Document Analyzer and click "Check Server Status"

Troubleshooting

If you're having trouble connecting to the server:

  1. Make sure Node.js is installed on your computer
  2. Check that all required packages are installed:
    npm install in the server directory
  3. Check for any error messages in the terminal
  4. Make sure no other application is using the same port
